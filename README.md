# WhatsApp Clone UI (Non-Functional)

This project is a static WhatsApp-like interface created using HTML and CSS.

## Objective
To practice UI design and layout structure of a chat application.

## Features
- Chat list panel
- Chat window with messages
- Input box and send button
- WhatsApp-style colors

## Technologies Used
- HTML
- CSS

## Note
This project is non-functional and does not include real messaging logic.
